영어학원에서 아르바이트 했던 경험을 살려 만든
학생관리 프로그램입니다.

Python의 내장 GUI인 Tkinter로 인터페이스를 구현했고,
DB는 MYSQL을 사용했습니다.

더 자세한 내용은 발표 pdf와 기능시연 영상을 참고해주세요.
